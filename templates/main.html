Welcome to free-wiki!

A wiki is a website that allows users to directly edit its content and structure while in the browser. free-wiki is a very simple wiki - any page with the url

    www.free-wiki.appspot.com/w/[name]

where [name] consists of lower-case letters, digits, and dash symbol, may be created and edited using HTML with no restrictions. Users may view the complete history of each page and any version can be edited and restored at any time. All pages are listed in the index.

The edit history of each user can be found at

    www.free-wiki.appspot.com/u/[name]

and all users are listed in the user index.

To become a user and create or edit pages please signup - your email address is not required.
